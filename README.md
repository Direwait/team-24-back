# Team24 Backend

Backend-приложение для системы управления заявками, построенное на Spring Boot.

##   Технологический стек

------
- **Java 21**
- **Spring Boot 3.5.4**
- **PostgreSQL 16+** + **Flyway** (миграции)
- **Spring Data JPA** + **Hibernate**
- **Spring Security** (аутентификация)
- **Spring Web** (REST API)
- **Maven** (сборка)
- **SMTP** (работа с почтой)

##  Предварительные требования

-------
Перед запуском убедитесь, что установлены:

1. **Java 21** ([скачать](https://jdk.java.net/21/))
2. **PostgreSQL 16+** ([скачать](https://www.postgresql.org/download/))
3. **Maven** ([скачать](https://maven.apache.org/download.cgi))

### Проверка установки:
``` bash
java -version
psql --version
mvn -version
````
### Настройка окружения
#### В .env настройте необходимую базу данных, например, как .env.template
````
    # Database
POSTGRES_DB=postgres
POSTGRES_USER=postgres
POSTGRES_PASSWORD=postgres
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
````
#### Также выберите Gmail почту для отправки всех уведомлений, ключ можно получить по
[ссылке](https://myaccount.google.com/) перейдя в "Пароли приложений"
````
MAIL=**********@gmail.com
MAIL_KEY=**** **** **** ****
````
### Запуск
````
mvn clean package
java -jar controller/target/controller-0.0.1-SNAPSHOT.jar
````

## Реализованный функционал

-----------------
### 🔹 Админ
- **Изменять существующий текст СОПД**
- **Изменять существующий шаблон письма кандидату**
### 🔹️ Менеджер

- **Заполнять базу запросов введя почту кандидата**

- **Фильтровать свои текущие запросы по статусам**
 ```` PENDING, REJECTED, APPROVED````

- **Автоматически отправлять письма на согласия СОПД кандидатам на любую почту**

- **Получать уведомления по почте при изменении статуса заявки**
### 🔹 Кандидат
- **Получает уникальную ссылку в письме**
- **Заполняет свои данные по маске**
- **соглашается/отказывается от СОПД**
